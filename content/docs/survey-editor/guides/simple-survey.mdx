# How to setup a simple survey

## Create a new survey

1. Go to **"Standalone editor"**.
2. Choose section **"Survey editor"**.
3. Click the **"Create new survey"** button and enter a survey key.

![Screenshot of survey editor welcome menu](images/welcomeEditor.png)

You are now ready to edit your new survey! 

![Screenshot of survey editor dashboard](images/dashboardEditor.png)

This dashboard shows you the content and structure of your survey:
- **Items**: This section lists all the items of the survey. Currently, no items have been added. You can click on an item to open it or drag and drop to reorder them.
- **Add a new item**: Click this option to create a new survey item.
- **Item ordering**: Here you can define whether the items should appear in the listed order or be shown in random order when the survey runs. Toggle the switch to enable random ordering.
- **Condition tab**: Switch to this tab to define conditional logic that controls when items should be shown based on participant responses.

## Add a single choice question 

1. Click the plus icon in the left toolbar or the **Add a new item** button in the **Items** tab.
2. Select the option **Question or Info item**.
3. Select the option **Single choice**. The single choice item is now shown in your item list. Click on the item to see this editor interface:

![Overview of single choice item content](images/overviewSinglechoice.png)

### üß≠ Top Navigation Bar

- **Survey Name:** Top left shows the name of the current survey.
- **Item Key:** You can change the item key by clicking on it and entering a new key.
- **Add Item Label:** Enter an item label for easier identification.
- **Three-Dot Menu (Top Right):** Provides besides normal mode additional actions for the current item:
    - `Source code` ‚Äì View or edit the raw JSON structure of the item.
    - `Copy` ‚Äì Duplicate the item.
    - `Move to other group` ‚Äì Move item to another group of questions.
    - `Delete item` ‚Äì Permanently remove the item from the survey.
- **Tabs:**
    - `Content` ‚Äì active tab where item text, options and structure is edited.
    - `Condition` ‚Äì define logic for when the item should be shown.
    - `Validation` ‚Äì add rules to validate user responses.
    - `Advanced` ‚Äì set advanced options such as confidential mode or prefill rules.

### üìù Content Section (Left Side)

#### **Header**
Configure the item header elements:
- **Title (`H1`)** ‚Äì Main question or prompt.
- **Subtitle (`H2`)** ‚Äì Additional explanation or secondary text.
- **Help/Info popup** ‚Äì Optional tooltip for additional guidance.

#### **Body**
Main content that surrounds the response options:
- **Top content (before response)** ‚Äì Text shown above the answer options.
- **Response options** ‚Äì Here you define the choices for the single choice question.
- **Bottom content (after response)** ‚Äì Text shown after answer options.

#### **Footnote**
- Additional information displayed at the bottom of the item.

### üîç Preview Section (Right Side)

- Shows a live **preview** of the item layout.
- **Language Switch:** Toggle between available languages (e.g., `nl` for Dutch and `en` for English).
